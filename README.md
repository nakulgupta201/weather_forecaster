# Weather Forecaster

This is a simple weather forecasting website made using HTML, CSS and JavaScript.

Users can search the weather of any city and also save their favourite cities.

## Features

- Search weather by city name
- Shows temperature and weather condition
- Add cities to favourites
- Delete favourite cities
- Separate page for favourite cities

## Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeather API
- LocalStorage

## How to Run

1. Download the project
2. Open the folder
3. Run `index.html` in your browser
